Commands and Instructions

  • Image Creation Command: /image create (link) (scale number)
  • Custom Discs Setup:
    • Install and set up the Simple VC Mod.
    • Upload an MP3 file to Discord.
    • Right-click the file, click "Copy Link."
    • Use the command: /cd download <discord link> <filename.extension>
  • Create Disc for Downloaded File:
    • Hold a disc in your hand.
    • Use the command: /cd create <filename.extension> "song title"
  • Convert YouTube to MP3: Use https://cobalt.tools/ to get the MP3.
  • Voice Chat Mod Download:
    • For Fabric: Download Fabric Mod
    • For Forge: Download Forge Mod
  • Testing Designs in Creative Mode: /mv tp creative
    To return: /mv tp world
  • Inspect Containers: /co inspect
    Click on the container to see who has taken items.
  • Set Custom Nickname: /nick
    To set a custom color, use: /nickcolor <hexvalue>
    Find your preferred color at RGB Color Picker.
  • Register as Premium Player: /premium
    Use this command if you own Minecraft to avoid having to log in with AuthMe every time.
